A secure code bin for privacy conscious engineering teams

Install Source Code
Trustb.in is a secure, open-source alternative to Github gist for modern engineering teams who care about privacy. Code snippets are securely encrypted and the whole system can be run self hosted.
Keep your code snippets yours

For a long time, developers have relied on Gist from Github. But now with co-pilot, its more imporant than ever to keep control over your teams ideas, notes and code.

Built for the modern cloud

Trustbin runs on modern cloud infrastructure like the DigitalOcean App Platform, or anywhere using Docker Compose

Encrypts Everything

Trustb.in encrypts your code data so that only your team can access it, ever. Feel secure hosting on public cloud infrastructure knowing your private details are encrypted so only your team can see them.
